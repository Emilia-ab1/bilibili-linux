# BILIBILI FOR Linux

特点：

1. 没有讨厌的标题栏
2. 支持小分辨率屏幕全屏
3. 检查更新支持

## 支持情况

- [x] 退出功能
- [x] 直播页全屏
- [x] 去除顶部边框
- [x] 检查更新

## 使用方法

1. 拉取代码
   ```
   git clone https://github.com/msojocs/bilibili-linux.git
   cd bilibili-linux
   ```
2. 安装
   ```
   tools/setup-bilibili
   ```
3. 启动
   ```
   bin/bilibili
   ```
